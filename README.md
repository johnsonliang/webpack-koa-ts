# webpack-koa-ts

> all in ts

使用ts-node启动koa2开发脚手架

## Build Setup

``` bash
# install dependencies
npm install (yarn)

# serve with hot reload at localhost:8080
npm run dev (yarn dev)

# build for production with minification
npm run build (yarn build)
```
